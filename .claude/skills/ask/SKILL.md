# Ask

## Workflow

### 1. 関連トピック検索

```bash
mise run fm-dump
```

出力JSONから、質問に関連するトピックを tags とタイトルで特定する。

### 2. 内容読み込み

該当する `topics/{topic}/README.md` を Read ツールで読み込む。
複数トピックにまたがる質問は、関連する全トピックを読む。

### 3. 回答

- トピック内のソースを引用して回答する
- トピックに記載がない情報は「既存の調査に含まれていない」と明示する
- 追加調査が有益な場合は survey スキルの利用を提案する

### 4. 該当なしの場合

fm-dump に該当がなければその旨を伝え、ユーザーが望めば survey スキルで新規調査を開始する。
